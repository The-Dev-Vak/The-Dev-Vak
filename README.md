## 🔨 What I'm building

**[latchkey.dev](https://latchkey.dev)** — I co-founded it and run it as COO. It watches your
GitHub Actions and answers the three questions a platform engineer would otherwise answer by
hand: where the money is going, what security regressions just landed, and which steps are
slow or flaky. Then it opens the PR.

We kept watching the same thing happen at startups and at enterprises — nobody owned the
pipeline, so the pipeline owned everyone.

Some of what we've built is open source:

| repo | what it is |
| --- | --- |
| [**CI-Doctor**](https://github.com/latchkey-dev/CI-Doctor) | an agent skill that teaches your AI to diagnose and durably fix failing pipelines |

| [**runner-benchmarks**](https://github.com/latchkey-dev/runner-benchmarks) | the numbers, published |
